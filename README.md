# UrbanAI 3D İstanbul

İBB açık verileri kullanılarak İstanbul'daki kentsel hizmetlerin
erişilebilirliğini analiz eden, yapay zekâ destekli ve üç boyutlu
web tabanlı karar destek sistemi.

## Projenin Amacı

Bu projenin amacı, İstanbul'daki kentsel hizmetlerin mekânsal
dağılımını incelemek, hizmet erişimi açısından dezavantajlı
bölgeleri belirlemek ve sonuçları etkileşimli bir harita üzerinde
göstermektir.

## Planlanan Özellikler

- İBB açık verilerinin incelenmesi ve temizlenmesi
- Kentsel hizmet noktalarının harita üzerinde gösterilmesi
- Mahalle bazında erişilebilirlik analizi
- Makine öğrenmesiyle benzer mahallelerin gruplandırılması
- Hizmet ihtiyacı yüksek bölgelerin belirlenmesi
- Sonuçların üç boyutlu web haritasında görselleştirilmesi
- FastAPI tabanlı veri servisinin geliştirilmesi

## Planlanan Teknolojiler

- Python
- Pandas
- GeoPandas
- Scikit-learn
- FastAPI
- Folium veya Leaflet
- CesiumJS
- Git ve GitHub

## Veri Kaynağı

Projede İBB'nin kamuya açık veri kaynaklarının kullanılması
planlanmaktadır.
